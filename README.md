## 消息推送

需要配置消息推送参数才可以收到消息通知，目前接入的消息通知是免费的，获取方法参考以下推送平台。

1. `pushplus` 需配置 token, token 可以去 [pushplus 官网](http://www.pushplus.plus)微信扫码登录获取
2. `server 酱` 推送即将废弃

* [pushplus 官网](http://www.pushplus.plus)
* [server 酱](https://sct.ftqq.com)
